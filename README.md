# Global Oil Prices & Geopolitical Events(2010-2026)
This repository contains my attempt to analyze and get a better understanding of the current state of the global oil market. The dataset was obtained from [https://www.kaggle.com/datasets/kavyadhyani/global-oil-prices-and-geopolitical-events/data].

# Goal
The main dataset of focus is the `geopolitical_events_timeline.csv`. The goal is to answer questions such as:
1. Are there more events occurring now, as compared to before?
2. What event type was the most common?
3. What countries/parties were involved in the most events?
4. What severity score was the most common?
